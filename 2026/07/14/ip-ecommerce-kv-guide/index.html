IP电商KV制作指南
📌 什么是IP KV？
IP KV（Key Visual）是品牌IP形象与活动主题结合的营销主视觉海报，用于电商活动、品牌宣传等场景。

🎯 快速开始
方式一：飞书表格提交任务

  1. 打开设计任务管理表格
  2. 新建记录，填写以下信息：
    任务名称：活动名称
    IP角色：选择要使用的IP（如喜猫、芒果仔）
    主题/场景：活动主题描述
    尺寸比例：3:4 或 9:16
    风格参考图：上传参考图（可选）
  3. 状态改为「待策划」
    方式二：直接对话
    直接告诉设计师需求，例如：
    “用喜猫做一个夏日露营主题的KV”
    “批量生成芒果仔4个场景的KV”
    “同时生成喜猫和云一朵的KV，都是音乐主题”
    📋 标准流程
    四阶段流程
    阶段1 - 需求创建：用户在表格填写基础信息 → 状态：待策划
    阶段2 - 策划方案：设计师给出2-3个文案+风格方案 → 状态：待确认
    阶段3 - 审核确认：用户选择方案 → 状态：确认生图
    阶段4 - 执行生成：设计师生成图片 → 状态：已完成
    💬 对话触发关键词
    基础生图
    “用XX IP做海报”
    “制作电商IP KV”
    “吉祥物KV”
    并发生图
    “一次生成4张” → n参数模式
    “多个场景” / “不同场景” → 多场景并发
    “同时生成IP1和IP2” → 多IP并发
    示例
    用户: “批量生成喜猫的KV，4个场景：草地露营、帐篷近景、泳池边、海边”
    用户: “这个场景帮我生成4张变体”
    用户: “同时生成喜猫和芒果仔的KV，都是夏日主题”
    📐 常用尺寸
    4K 3:4 → 2448×3264 或 1024×1366（比例3:4）
    4K 9:16 → 2160×3840（比例9:16）
    2K 16:9 → 2048×1152（比例16:9）
    4K 1:1 → 2864×2864（比例1:1）
    🎨 风格选择参考
    ST0011 Sunny 3D Avatar — 阳光户外、度假氛围
    ST0070 软胶黏土3D卡通 — 可爱童趣、露营主题
    ST0148 Bold Monoline Mascot — 简洁线条、极简风格
    📁 可用IP列表
    IP001 古茗云岭茉莉 — 古茗 — 商品IP
    IP002 莉莉 — 个人IP — 活泼可爱
    IP003 芒果仔 — 芒果TV — 品牌公仔
    IP004 马奇 — 个人IP — 时尚风格
    IP005 猕猴仔 — 爱奇艺 — 品牌IP
    IP006 云一朵 — 百度网盘 — 角色IP
    IP007 大耳狗 — 三丽鸥 — 角色IP
    IP009 哔哩哔哩 — 哔哩哔哩 — 角色IP
    IP011 喜猫 — 喜马拉雅 — 角色IP
    ⚠️ 注意事项
    IP一致性
    • 生成时会使用IP参考图保证形象一致性
    • 不要在描述中写IP的外貌特征（肤色、五官等）
    文字要求
    • 主标题：请提供完整文案
    • 活动时间：如需展示，请明确日期格式
    • 品牌logo：可选择在画面角落展示或不使用
    反馈调整
    • 生成后可提出调整需求（颜色、构图、元素等）
    • 多次优化直到满意为止
